The following chips are used on the RoboCube v2.0 board:
  • Processor board
    • main CPU MC68332
    • 29F800 512Kx16 Flash EPROM
    • 68400 512Kx8 SRAM
    • Power watchdog
  • Busmaster board
    • MC68681 DUART
    • RS-232 level shifter
    • I2C busmaster
    • 74abt652 Bus exchanger
    • 74als512 Comparator
    • GAL22V10
    • 74abt16823 Binary outputs
  • IO board
    • PCF 8574 I2C 8x binary I/O
    • PCF 8591 I2C 4x analog IN, 1x analog OUT
  • FPU and DRAM board
    • 48C8000 8Mx8 DRAM
    • 74abt32316 Universal bus exchanger (MUX)
    • 74hct4020 Ripple Counter
    • MC68882 FPU
