The following chips are used on the RoboCube board:
  • Processor board
    • main CPU MC68332
    • 27C512 64Kx8 EPROM
    • 681000 128Kx8 SRAM
    • Power watchdog
  • Busmaster board
    • 68681 DUART
    • RS-232 level shifter
    • PCF 8584 I2C busmaster
  • IO board
    • PCF 8574 I2C 8x binary I/O
    • PCF 8591 I2C 4x analog IN, 1x analog OUT
